## 介绍 ##
#### 云库能做些什么
* 统一的云端资源管理
* 针对云服务器运维的访问优化
#### 目前为止，云库可以提供以下支持：
* 资产统一管理
  * 云资产的在线共享
* 云服务器管理
  * 服务器接入面板快速访问
